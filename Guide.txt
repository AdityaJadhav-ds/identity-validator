# 🚀 Identity Validator – Setup & Run Guide

## 📁 Project Structure

identity-validator/
│
├── backend/   → FastAPI server
└── frontend/  → React app

---

## ⚙️ Prerequisites

Make sure you have installed:

* Python (3.10+)
* Node.js (v16+)
* npm
* Git (optional)

---

## 🔧 STEP 1 — Setup Backend

### 1. Go to backend folder

cd backend

### 2. Create virtual environment (if not already)

python -m venv venv

### 3. Activate virtual environment

👉 Windows:
venv\Scripts\activate

👉 Mac/Linux:
source venv/bin/activate

---

### 4. Install dependencies

pip install -r requirements.txt

---

### 5. Run backend server

uvicorn app.main:app --reload

---

### ✅ Backend will run at:

http://127.0.0.1:8000

### 📄 API Docs:

http://127.0.0.1:8000/docs

---

## 💻 STEP 2 — Setup Frontend

### 1. Open new terminal

cd frontend

### 2. Install dependencies

npm install

---

### 3. Create `.env` file inside frontend folder

Add this line:
REACT_APP_API_URL=http://127.0.0.1:8000

---

### 4. Start frontend

npm start

---

### ✅ Frontend will run at:

http://localhost:3000

---

## ▶️ HOW TO USE

1. Enter:

   * Full Name
   * Aadhaar Number
   * PAN Number

2. Upload:

   * Aadhaar document (PDF/Image)
   * PAN document (PDF/Image)

3. Click **Verify Identity**

4. System will:

   * Extract text (OCR)
   * Validate data
   * Show result (Verified / Issues)

---

## 📸 IMPORTANT (FOR BEST RESULTS)

Use:

* Clear images or PDFs
* No blur or shadow
* Full document visible
* High resolution

Avoid:

* WhatsApp compressed images
* Cropped documents
* Tilted photos

---

## ⚠️ COMMON ISSUES

### ❌ API not working

Make sure backend is running

---

### ❌ Module errors

Activate virtual environment again

---

### ❌ Slow response

OCR processing may take a few seconds (normal)
